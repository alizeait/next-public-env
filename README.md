# next-public-env

Use both runtime and build environment variables in both the server and the client in your Next.js app

## Installation

```bash
pnpm add next-public-env
```

Or using other package managers:

```bash
npm install next-public-env
# or
yarn add next-public-env
# or
bun add next-public-env
```

## Usage

```typescript
import { example } from 'next-public-env';

example();
```

## Development

```bash
# Install dependencies
pnpm install

# Run tests
pnpm test

# Build
pnpm build

# Check types
pnpm check-types

# Lint
pnpm lint

# Format
pnpm format
```
